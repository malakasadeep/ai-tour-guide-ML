# Shared Resources

This folder contains data assets shared across multiple pillars.

## Files

### locations_metadata.csv
Master list of 80 Sri Lankan tourism locations.

| Column | Description |
|--------|-------------|
| `name` | Location name |
| `l_hist` | Historical significance (0-1) |
| `l_adv` | Adventure level (0-1) |
| `l_nat` | Natural beauty (0-1) |
| `l_rel` | Religious significance (0-1) |
| `l_outdoor` | Outdoor activity (0/1) |
| `l_lat` | Latitude |
| `l_lng` | Longitude |

## Coverage (9 Regions)
